# SK Aluminium
### Glass & Aluminum Business Management System

![Version](https://img.shields.io/badge/Version-1.0.0-blue)
![Status](https://img.shields.io/badge/Status-Frontend_Complete-success)

ระบบบริหารจัดการสต็อกและคำนวณต้นทุนโครงการ สำหรับธุรกิจติดตั้งกระจกและอลูมิเนียม เน้นความเรียบง่ายและใช้งานได้จริงบนทุกอุปกรณ์

## 🎯 Key Features
* **Inventory**: จัดการคลังวัสดุและเศษวัสดุ
* **Projects**: คำนวณต้นทุน กำไร และออกใบเสนอราคา
* **Attendance**: ระบบบันทึกเวลาเข้า-ออกงานพนักงาน
* **Roles**: แยกการเข้าถึงระหว่าง CEO และ Employee

## 🏗️ Technical Stack
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
* **Design**: Minimalist UI, Responsive Design
